# Preorder Manager

A full-stack preorder management application built with Next.js 16, Prisma, and SQLite.

## Features

- ✅ List preorders with filters (All, Active, Inactive)
- ✅ Sort by Name, Products, Starts At, and Date
- ✅ Pagination
- ✅ Create new preorders
- ✅ Update existing preorders
- ✅ Delete preorders
- ✅ Toggle Active/Inactive status
- ✅ Bulk selection checkboxes
- ✅ Loading states
- ✅ Responsive UI matching the provided designs

## Tech Stack

- Next.js 16 (App Router)
- Prisma ORM
- SQLite Database
- Tailwind CSS
- TypeScript
- Zod for validation
